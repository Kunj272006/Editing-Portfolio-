# 🎬 Video Editor Portfolio Website

A modern, interactive **video editor portfolio website** built using HTML, CSS, and JavaScript. This project showcases creative video editing work with a cinematic UI/UX, smooth animations, and a professional layout.

---

## 🚀 Features

- 🎨 **Multiple Themes**  
  Dark, Light, and Neon modes for a dynamic visual experience.

- 🎥 **Featured Work Section**  
  Displays best video edits with clickable previews linked to Google Drive.

- 💼 **Client Work Showcase**  
  Dedicated section highlighting real client projects and results.

- 📈 **Stats & Experience**  
  Shows number of videos delivered, satisfaction rate, and experience.

- 🔁 **Journey Section (Then → Now)**  
  Visual comparison between first edit and latest work.

- 🧠 **Skills Section**  
  Displays editing skills with progress indicators.

- 📬 **Contact Section**  
  Includes form and social links for client communication.

- ✨ **Advanced UI Effects**
  - Smooth animations
  - Glassmorphism design
  - Hover effects
  - Custom cursor with gooey effect
  - Animated background canvas

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Advanced Styling & Animations)  
- JavaScript (DOM Manipulation & Effects)  
- Google Fonts  
- Google Drive (for video hosting)
